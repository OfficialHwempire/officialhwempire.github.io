# Mockito 에 대해 알아보자


## 1. Mockito 란?

Mockito(모키토)는 자바 테스트에서 사용하는 Mocking 프레임워크입니다.
Mock을 만들어 테스트를 편리하게 만드는 라이브러리입니다.

## 2. Mockito Stub

 - Mock 의 기능을 모아넣은 Mockito 에서 Stub은 미리 정해진 값을 반환하는 객체
 - 순수 로직을 테스트 할 때 Stub 사용

```java
given(userRepository.findById(id))
    .willReturn(Optional.of(user));

```

## 3. Mockito spy

- 실제 객체의 일부분을 Mocking 한 가짜 객체
- 테스트 불가능한 구조를 부분적으로 제어할 때 사용
- 레거시 코드 테스트 

```java
UserService spyService = spy(realService);
```
## 4. Mockito Mock

- 행위를 검증하는 가짜 객체
- 협력 객체와 상호 작용 확인 하는 용도

```java

verify(messageRepository).save(any());

```
